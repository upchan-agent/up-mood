# 🆙 UP Mood

**Universal Profile Ecosystem Attributes Visualizer**

Discover your Universal Profile's ecosystem attributes and species based on on-chain activity.

---

## 🌐 Live Demo

https://up-mood.vercel.app

---

## ✨ Features

- **🌱 Ecosystem Attributes**: 5 attributes calculated from your transaction history
  - 💰 **Wealth** - LYX and token transfers
  - ⚡ **Vitality** - General execute operations
  - 🧠 **Intelligence** - Complex contract executions
  - 🎨 **Creativity** - Profile metadata updates (LSP-3, LSP-12)
  - 🤝 **Sociability** - Follow/endorse and permission management (LSP-6, LSP-26)

- **🏷️ Species System**: Your dominant attribute determines your species
  - 🐣 Baby - Just getting started
  - 🦊 Merchant - Wealth-focused
  - 🦁 Warrior - High activity
  - 🦉 Scholar - Intelligence-focused
  - 🦋 Artist - Creativity-focused
  - 🕊️ Diplomat - Sociability-focused
  - 🦄 Explorer - Balanced attributes

- **🎨 Modern Design**: Beautiful gradient UI with smooth animations

---

## 🚀 Usage

1. Open the app
2. Enter your Universal Profile address (or connect via Grid)
3. Click "Analyze"
4. View your ecosystem attributes and species!

---

## 🛠️ Tech Stack

- **React 19** + **TypeScript**
- **Vite** - Build tool
- **@lukso/up-provider** - Universal Profile connection
- **LUKSO Blockscout API** - On-chain data
